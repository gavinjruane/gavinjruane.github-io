Spanning Sets

If \(y_1\) and \(y_2\) are solutions of a differential equation, so is \( c_1y_1+c_2y+2 \) for \(c_1\),\(c_2\in \mathbb{R}\). This is a linear combination.

In a vector space \(V\), the set of linear combinations for any subset of vectors is a span.

\[ \text{span}\{\overrightarrow v_1, \overrightarrow v_2, \ldots,\overrightarrow v_n \}= \{c_1\overrightarrow v_1 +c_2\overrightarrow v_2+\ldots +c_n\overrightarrow v_n \} \]

If a set "spans \(V\)", it captures every vector in \(v\).

\[ \text{span}\{\overrightarrow v_1, \overrightarrow v_2, \ldots, \overrightarrow v_n \}=V \]
