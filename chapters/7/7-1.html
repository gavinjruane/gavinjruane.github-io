Key ideas

Fundamental idea

\[\lim_{x \to a}f(x)=L\]

The limit as \(x\) approaches \(a\) on \(f(x)\) is \(L\).

  • Can exist and can be or cannot be reached
  • A zero over zero (\(\frac{0}{0}\)) will require the usage of L'hopital' s rule.

Steps for evaluating a L'hopital's rule problem:

  1. Substitute the given \(x\) value into the limit problem and see if it results in a non-zero answer.
  2. If the result is a non-zero answer: the number is the answer to the problem.
  3. If the result is \(\frac{0}{0}\): take the derivative of the numerator and then the denominator. Do not use the quotient rule.
    1. Re-substitute the given \(x\) value into the function. Repeat these steps as many times as necessary.

Sample problems

  1. \(\displaystyle{\lim\limits_{x \to 0}\frac{\sin x}{x}}\)
    2. \[\begin{align} &=\frac{\sin 0}{0} \\ &=\frac{0}{0} \end{align}\]

    We got a \(\frac{0}{0}\). We'll need to use L'hopital's rule to solve the rest of this problem.

    \[\begin{align} f'(x)&=\frac{-\cos x}{1} \\ &=\frac{-\cos 0}{1} \\ &=1 \end{align}\]
  2. \(\displaystyle{\lim\limits_{x \to 2}\frac{1-e^{x-2}}{x^2-4}}\)
    3. \[\begin{align} &=\frac{1-e^{2-2}}{2^2-4} \\ &=\frac{0}{0} \end{align}\]

    We got a \(\frac{0}{0}\). We'll need to use L'hopital's rule to solve the rest of this problem.

    \[\begin{align} f'(x)&=\frac{-e^{x-2}\times 1}{2x} \\ &=-\frac{1}{4} \end{align}\]
  3. \(\displaystyle{\lim\limits_{x \to-3}\frac{x^2+6x+1}{x^2+9}}\)
    4. \[\begin{align} &=-\frac{8}{18} \\ &=-\frac{4}{9} \end{align}\]

    This limit was all good without L'hopital's rule, so it was as simple as substituting and simplifying.

  4. \(\displaystyle{\lim\limits_{x \to 2}\frac{x^2+4x+4}{x^3-x^2-8x+12}}\)
    5. \[\begin{align} &=\frac{(2)^2+4(2)+4}{(2)^3-(2)^2-8(2)+12} \\ &=\frac{0}{0} \end{align}\]

    We got a \(\frac{0}{0}\). We'll need to use L'hopital's rule to solve the rest of this problem.

    \[\begin{align} f'(x)&=\frac{2x-4}{3x^2-2x-8} \\ &=\frac{2(2)-4}{3(2)^2-2(2)-8} \\ &=\frac{0}{0} \end{align}\]

    Oh no...\(\frac{0}{0}\) again! Try L'hopital's rule yet again.

    \[\begin{align} f''(x)&=\frac{2}{6x-2} \\ &=\frac{2}{6(2)-2} \\ &=\frac{1}{4} \end{align}\]
