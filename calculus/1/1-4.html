Key ideas

Quotient rule

\[\begin{align} \frac{d}{dx}\left[\frac{u}{v}\right]=\frac{v\cdot\frac{du}{dx}-u\cdot\frac{dv}{dx}}{v^2} \end{align}\]
  • Check for points of exclusion before using the quotient rule, and if there are any, remove them.
  • Simplify the overall function if you can before using the steps for the quotient rule.

Steps for using the quotient rule:

  1. Identify if there is a quotient between two functions and if there is, label \(U\) and \(V\).
  2. Find and simplify the derivatives of \(U\) and \(V\).
  3. Substitute in all values and terms.
  4. Simplify the derivative using various techniques like common factors, common denominators, reduing, etc.

Sample problems

  1. \(\displaystyle{\frac{d}{dx}\left[\frac{6x}{x^2+4}\right]}\)

    2. Finding individual derivatives

    \[\begin{align} u&=6x\to 6 \\ v&=x^2+4\to 2x \end{align}\]

    Placing parts together

    \[\begin{align} \frac{dy}{dx}&=\frac{(x^2+4)(6)-(6x)(2x)}{(x^2+4)^2} \\ &=\frac{6x^2+24-12x^2}{(x^2+4)^2} \\ &=\frac{24-6x^2}{(x^2+4)^2} \end{align}\]
